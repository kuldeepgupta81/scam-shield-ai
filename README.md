# 🚨 Scam Shield AI

AI-powered scam detection web app that analyzes suspicious messages, URLs, and screenshots to identify potential scams.

---

## 🔥 Features

* 🔍 **Message Scam Detection**
  Detects OTP scams, financial fraud messages, and suspicious patterns.

* 🔗 **Phishing URL Scanner**
  Checks links for common phishing indicators.

* 📷 **Screenshot Scam Detection**
  Upload WhatsApp or SMS screenshots to analyze possible scams.

* 📊 **Dashboard Analytics**
  Displays scan statistics and recent scan history.

* ⚡ **Real-time Risk Scoring**
  Generates a risk score based on detected scam indicators.

* 🎨 **Modern Premium UI**
  Dark cyber-style interface built with Tailwind CSS.

---

## 🛠 Tech Stack

* **Next.js** – React framework for full-stack apps
* **React** – UI library
* **Tailwind CSS** – Utility-first styling

## 🚀 Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/yourusername/scam-shield-ai.git
cd scam-shield-ai
npm install
npm run dev
```

Then open:

```
http://localhost:3000
```
